# ThemeKit.Swift

UI themes and components

## Requirements

* Xcode 15.4+
* Swift 5.10+
* iOS 14.0+

## Installation

### Swift Package Manager

```swift
dependencies: [
    .package(url: "https://github.com/sunimp/ThemeKit.Swift.git", .upToNextMajor(from: "2.3.0"))
]
```
